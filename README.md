# Brain Tumor Classification System 🧠

## 📌 Project Description

This project is a web-based application that predicts brain tumor types using deep learning.

## 🚀 Features

* Upload MRI image
* Predict tumor type (glioma, meningioma, pituitary, no tumor)
* Display results with accuracy

## 🛠️ Technologies Used

* Python
* Flask
* TensorFlow / Keras
* HTML, CSS

## 📂 Project Structure

* Training & Testing datasets
* Model file (.h5)
* Flask app (app.py)
* Templates (HTML pages)

## ▶️ How to Run

1. Install dependencies
2. Run:
   python app.py
3. Open browser → http://localhost:5000
